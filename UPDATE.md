# Instructions de mise à jour vers la version 1.3.0

Date de publication: 2025-03-22

## Comment mettre à jour

1. Récupérez les derniers changements:
   ```bash
   git fetch origin
   git checkout v1.3.0
   ```

2. Installez les dépendances:
   ```bash
   npm install
   ```

Pour connaître les changements apportés par cette version, consultez le [CHANGELOG.md](./CHANGELOG.md).
